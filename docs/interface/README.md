# Interface Documentation

This directory contains documentation for Mapper's public interfaces.

## Contents

- [CLI Commands](cli.md) - Command-line interface reference
- [Python API](api.md) - Public Python API reference (planned)

## Purpose

Interface documentation describes **how to use** Mapper's public APIs:
- CLI commands with arguments, options, and examples
- Python function/class signatures with usage examples
- API endpoints with request/response schemas

**This documentation must be kept up to date with code changes.**

## When to Update

Update interface documentation when:
- Adding new CLI commands or options
- Modifying command signatures or behavior
- Adding/changing public Python APIs
- Changing API endpoints or schemas

## Format

Each interface document should include:
1. **Overview** - What the interface does
2. **Usage** - How to use it with examples
3. **Parameters** - Arguments, options, or fields
4. **Return Values** - What the interface returns
5. **Examples** - Real-world usage examples
6. **Related** - Links to user journeys and technical docs
